
Granite Data Services Java Client (http://www.graniteds.org)
-------------------------------------------------------------------------------

* Provide:

  A Java client library and framework that may be used to connect to a
  remote GraniteDS server.

* Requirements:

  JDK 6+, JavaFx 2+ (optional).
  
* Build:

  Download the complete source bundle and import it as an Eclipse project.
  
  Run the build.xml file.
